Tema 4 - Grundlæggende Animation

I tema 4 er vi blevet introduceret til skitsering og udvikling af ide og visuelt koncept. Her har vi arbejdet med aktivets diagrammer og State machine diagrammer, samt figur- og spil- elementer, baggrundsdesign, UI-elementer, typografi, farve og komposition, hvilket vi har udviklet i Adobe Illustrator. Ydermere har vi arbejdet med udviklingen af vores CSS viden, da vi har arbejdet med CSS animation, samt er vi blevet introduceret til JavaScript. Dette afsluttede vi ved temaopgaven ”04.04.01” hvortil vi udarbejdet overstående i en dokumentations-website med link til eget spil.

Opgavebeskrivelse

I dette animationstema skulle vi ende ud med en dokumentations-website med link til eget spil under opgaven ”04.04.01”. Processen startede med idé og design udviklingsfasen hvor vi udarbejdet papir prototype samt aktivitetsdiagrammer for spillet. Vi rentegnede figurer og skærme i Adobe Illustrator. Efter designprocessen fortsat vi til at starte kodningen af spillet ved brug af HTML, CSS og JavaScript, hertil udviklede vi et state machine diagram. Der er i spillet gjort brug af CSS-positioner og transform, samt CSS-animationer og videreudvikling af disse via JavaScript. I JavaScript arbejdede vi med bl.a. random position, animationend, anitioniteration reflow.

Til slut samlede vi alt dokumentation fra projektet og udviklede en website, med direkte link til spillet.
