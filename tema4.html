Tema 4 - Grundlæggende Animation

I tema 4 blev vi introduceret til skitsering og udvikling af ide og visuelt koncept. Vi arbejdede med aktivetsdiagram og state machine diagram, vi udviklede figur- og spil- elementer, baggrundsdesign, UI-elementer, typografi, farve og komposition, i adobe illustrator. Hertil videreudviklede vi vores viden af CSS, da vi arbejdede med CSS animation, samt blev introduceret til JavaScript. Til den afsluttende temaopgave ”04.04.01” udarbejde jeg et dokumentations-website med link til eget spil.

Opgavebeskrivelse

Opgave 04.04.01 havde animation som tema. I dette tema var formålet at udarbejde et dokumentations-website med link til eget spil. Første fase i denne proces var idé og design. I denne fase lavede jeg en papirsprototype samt et aktivitetsdiagram for mit spil. Jeg rentegnede figurer og skærme i adobe illustrator. Efter designprocessen startede jeg kodningen af spillet ved brug af HTML, CSS og JavaScript, undervejs lavede jeg et state machine diagram. I spillet gjorder jeg brug af CSS-positioner og transform, samt CSS-animationer og videreudvikling af disse via JavaScript. I JavaScript arbejde jeg med random position, animationend, animationiteration reflow.
