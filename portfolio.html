TEMA 2 - Grundlæggende Web

Under tema 2 blev vi introduceret for grundlæggende redskaber indenfor design og kodning. Fagelementerne blev implementeret gennem en række delopgaver som til sidst endte ud i den afsluttende temaopgave ”02.02.01 emnesite”. Denne opgaver er herved første opgave med brug af grundlæggende kode og designprocesser. Opgave dannede fundament for vores videre udarbejdelse af HTML og CSS i Visual Studio Code. Ydermere blev vi introduceret til praksisnære principper og metoder til design af brugergrænseflader. Vi arbejdede med programmer som Adobe Photoshop og XD til designprocesserne.

TEMA 3 - Grundlæggende UX

Ved dette tema har vi beskæftiget os med UX’erens rolle på tværs af designdiscipliner og en iterative udviklingsproces. Vi har haft fokus på research, analyse og indholdsproduktion om brugere og brugssituationer via desk research, serveys, obeservation og interviews, samt gennem viden om UI-konventioner, prototyping og testing på relevante brugere. Dette resulteret i temaopgaven ”03.02.02 Løsning” som blev udarbejdet i programmet Abode XD, samt afsluttende ”03.03.02 pitch”.

TEMA 4 - Grundlæggende Animation

I tema 4 er vi blevet introduceret til skitsering og udvikling af ide og visuelt koncept. Her har vi arbejdet med aktivets diagrammer og State machine diagrammer, samt figur- og spil- elementer, baggrundsdesign, UI-elementer, typografi, farve og komposition, hvilket vi har udviklet i Adobe Illustrator. Ydermere har vi arbejdet med udviklingen af vores CSS viden, da vi har arbejdet med CSS animation, samt er vi blevet introduceret til JavaScript. Dette afsluttede vi ved temaopgaven ”04.04.01” hvortil vi udarbejdet overstående i en dokumentations-website med link til eget spil.
