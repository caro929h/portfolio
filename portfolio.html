TEMA 3 - Grundlæggende UX

I tema 3 beskæftigede vi os med UX’erens rolle på tværs af designdiscipliner og en iterative udviklingsproces. Vi havde fokus på research, analyse og indholdsproduktion om brugeren og brugssituationer, hvilket vi dannede viden om via desk research, serveys, observation og interviews. Samt gennem viden om UI-konventioner, prototyping og testing på relevante brugere. Til slut endte det ud i temaopgaven ”03.02.02 løsning” som blev udarbejdet i adobe XD, samt afsluttende præsentation ”03.03.02 pitch”.

TEMA 4 - Grundlæggende Animation

I tema 4 blev vi introduceret til skitsering og udvikling af ide og visuelt koncept. Vi arbejdede med aktivetsdiagram og state machine diagram, vi udviklede figur- og spil- elementer, baggrundsdesign, UI-elementer, typografi, farve og komposition, i adobe illustrator. Hertil videreudviklede vi vores viden af CSS, da vi arbejdede med CSS animation, samt blev introduceret til JavaScript. Til den afsluttende temaopgave ”04.04.01” udarbejde jeg et dokumentations-website med link til eget spil.
